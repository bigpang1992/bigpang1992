AI周刊第四期，2025年11月第一周。

Sora应用正式登陆泰国

10月30日，据AI News报道，泰国成为亚洲最早获得OpenAI Sora应用的国家之一。这意味着泰国用户可以直接通过手机App使用Sora来生成AI视频。

Sora的坎坷历程

回顾一下Sora的历程，你会发现这简直是AI领域"跳票之王"：

时间事件大众反应
2024年2月OpenAI首次展示Sora视频生成能力全网震惊：“电影完蛋了！”
2024年中一直没有开放使用“到底什么时候能用？”
2024年底Sora终于有限开放“就这？不如宣传片好”
2025年5月Google发布Veo 3与之竞争“Sora还没站稳脚跟就被追上了”
2025年7月Veo 3大范围开放“Google后发先至了？”
2025年10月Sora以独立App形式在亚洲上线“终于认真做产品了”

Sora选择泰国作为亚洲首批市场，可能跟几个因素有关：

  1. 泰国政府对AI技术的开放态度。此前泰国在数字经济方面一直走在东南亚前列，监管环境比较友好
  2. 创作者经济活跃。泰国的短视频和社交媒体创作者群体非常庞大，对视频生成工具有实际需求
  3. 试水意味。在正式进入更大的亚洲市场（日本、韩国等）之前，先在泰国试一试

对中国用户的现实

对于中国用户来说，Sora的官方渠道仍然不可用。不过国内已经有不少替代品在追赶：

  • 可灵（快手）：目前国产视频生成工具中口碑最好的
  • Vidu（生数科技）：清华系背景，对中国风场景理解好
  • 字节的视频生成项目：还在内测阶段，但字节的数据优势很大
  • 各种基于开源模型的方案：ComfyUI + AnimateDiff等

说实话，在视频生成领域，国产工具跟Sora的差距比文本AI领域跟ChatGPT的差距要小很多。主要原因是视频生成技术还在早期，大家都在摸索阶段，先发优势没那么大。

ChatGPT Atlas隐私争议开始发酵

上周发布的Atlas浏览器，这周开始遇到第一波质疑——隐私问题。

为什么Atlas的隐私问题比普通浏览器更严重？

想想看这个场景：

普通浏览器（Chrome）收集的数据：

  • 你访问了哪些网站
  • 你的搜索记录
  • 你的书签和历史记录
  • 基本的用户行为数据

Atlas收集的数据（理论上）：

  • 上面所有的 +
  • AI理解你在每个页面上做什么
  • AI分析你的阅读习惯和兴趣
  • AI可以理解你填写的表单内容
  • 如果用了Agentic Mode，它知道你的购物偏好、预算、联系信息……

区别在哪里？Chrome收集的是"原始数据"——一堆URL和点击记录，需要数据分析师来解读。Atlas收集的是"已理解的数据"——AI已经帮你解读好了。这是一个质的变化。

举个极端的例子：Chrome知道你访问了WebMD上关于某种疾病的页面。Atlas知道你**“在研究甲状腺问题的症状，可能是因为你或你的家人最近有相关症状”**。

虽然OpenAI表示Atlas有隐私保护机制，但考虑到ChatGPT本身就通过对话数据训练模型，用户的疑虑完全合理。

行业人士怎么看？

数据隐私专家们普遍持谨慎态度。主要担忧集中在：

  1. OpenAI的隐私政策是否足够透明
  2. 收集的数据是否会用于模型训练
  3. 如果OpenAI被收购/破产，这些数据怎么处理
  4. 不同国家和地区的数据合规问题（GDPR、PIPL等）

预计在未来几周，关于Atlas隐私问题的讨论会持续升温。这也可能成为Atlas能否被大规模采用的关键因素之一。

GPT-5生态持续扩展

在GPT-5发布近三个月后，围绕它的生态正在成型：

企业采用加速

越来越多的公司开始用GPT-5替代之前的AI解决方案。根据OpenAI官方数据，GPT-5 API在发布后3个月内的调用量已经超过了GPT-4o同期的2倍。

主要驱动力是GPT-5在以下领域的明显提升：

  • 复杂推理任务（比如多步骤的数据分析）
  • 代码生成和审查的准确性
  • 对超长上下文的保持能力

开发者体验改善

API的调试工具和文档都在快速迭代。OpenAI在10月份更新了开发者平台，加了几个实用功能：

  • Token Usage Dashboard — 可视化你的API调用量和成本
  • Playground升级 — 支持多轮对话测试和参数调优
  • Eval Framework — 帮助开发者评估模型在特定任务上的表现

一个有趣的"回暖"现象

很多用户在经历了最初对GPT-5的失望后，开始重新发现它的优势。这个过程有点像：

  1. 发布当天：“牛！新模型来了！”
  2. 使用一周后：“也没多大提升嘛……还不如GPT-4o好聊”
  3. 使用一个月后：“等等，它在某些任务上确实更强了”
  4. 使用三个月后：“已经回不去GPT-4o了，复杂任务真的差很多”

这个"U型曲线"在每次模型更新时都会出现。人们总是先关注"丢失了什么"，然后才慢慢发现"获得了什么"。

本周总结

Sora进入亚洲市场标志着AI视频生成开始从"技术演示"走向"大众消费"。而Atlas的隐私争议则提醒我们：AI产品在追求便利的同时，必须认真对待用户数据安全。

这两件事其实指向同一个趋势：AI正在从"可选的增强工具"变成"日常生活的基础设施"。当一个技术开始渗入日常生活时，隐私、安全、可靠性这些问题就不再是技术极客担心的事，而是每个人都需要关注的事。
