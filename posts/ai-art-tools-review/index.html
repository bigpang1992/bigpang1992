花了一周时间横评了6款AI绘画工具，结果出乎意料

📅 2026年03月08日 ⏱ 2 分钟阅读 🏷️ AI, AI绘画, Midjourney, Stable Diffusion, 工具测评
📢 推广
📢

广告位虚位以待，联系站长投放

上周闲得慌，把市面上主流的 AI 绘画工具都撸了一遍。本来想随便玩玩，结果一测就是一整周，直接上瘾了。

起因是我需要给博客做几张头图，以前都是去 Unsplash 找免费图片，但想找到跟文章主题完全贴合的图真的太难了。想着现在AI绘画这么火，干脆自己生成得了。结果一入坑就停不下来，六款工具挨个试了个遍。

这篇文章不搞那些虚的，直接上干货。每款工具我都用了同样的提示词测试，尽量做到公平对比。

测评选手

工具价格特点我的使用时长

Stable Diffusion 3.5免费/本地开源之光断断续续半年

Flux 1.1 Pro按量付费新锐黑马三个月

懒人总结

先说结论，省得你们往下翻：

  • 追求画质和美感 → Midjourney，没啥好说的，审美天花板
  • 要自由度和可控性 → SD 3.5 本地部署，ControlNet 一套下来随便折腾
  • 偶尔用用不想折腾 → DALL·E 3，ChatGPT 里直接聊就能画
  • 中文提示词写得多 → 通义万相，中文语义理解碾压海外选手
  • 性价比最高 → Flux 1.1 Pro，画质逼近 MJ，价格良心

文心一格嘛……怎么说呢，进步很大，但跟前面几位比还是差点意思。

测试方法

为了公平对比，我用了三组统一提示词来测试每款工具：

  1. 人物类：“一个穿着白色连衣裙的年轻女性，站在樱花树下微笑，日落时分，电影级光影”
  2. 场景类：“赛博朋克风格的中国古镇夜景，霓虹灯倒映在水面上，远处有现代高楼”
  3. 物品类：“一杯热拿铁咖啡放在旧木桌上，旁边有一本打开的书和一副圆框眼镜，暖色调，摄影风格”

每个提示词在每款工具上至少跑了5次，取最好的结果来比较。海外工具用英文版提示词，国产工具用中文版。

详细感受

Midjourney 依然是审美标杆

不得不承认，MJ 出图的"高级感"还是别家比不了的。同样的提示词，MJ 的构图、光影、色调就是比别的好看。就像有些人天生会穿搭，工具也一样。

v7 之后相比 v6 有几个明显进步：

  • 原生支持中文提示词了，虽然还是英文更准但至少能用了
  • 手指终于画对了（大部分时候），不会再出现六指琴魔
  • 文字渲染有进步，简单的英文单词基本能写对
  • --personalize 参数可以让风格更贴合你的审美偏好

我最常用的MJ参数组合：--ar 16:9 --style raw --stylize 200 --quality 2。--style raw 让出图更写实，不会太"MJ味"。--stylize 控制在100-300之间比较自然，太高就开始"过度美化"了。

缺点也很明显：

  1. 太贵了。$10/月基础版只有200张，重度用户很快就超了。按我这种测评强度，一周200张根本不够
  2. 精确控制是个灾难。让它画"左手拿苹果右手拿香蕉"，它能给你画出左手拿香蕉右手拿苹果还一脸得意。想要精确控制构图和元素位置，你会被气死
  3. 只能在Discord或网页端用。虽然有了网页版，但界面比较简陋，管理生成的图片不太方便
  4. 商用条款要注意。免费版生成的图不能商用，10刀版在年收入100万美元以下才能商用

适合人群：设计师、自媒体人、需要高品质配图的人。不差钱的话MJ是最省脑子的选择。

SD 本地部署真的折腾，但折腾完值了

我花了大半天在我那台 4070Ti（12GB显存）上部署 SD 3.5。整个过程大概是这样的：

  1. 装 Python 3.10 环境 → 20分钟
  2. 装 ComfyUI（我没用 WebUI，ComfyUI 的节点流更灵活） → 30分钟
  3. 下载 SD 3.5 Medium 模型（约 5GB）→ 看网速，我花了40分钟
  4. 装各种插件：ControlNet、IPAdapter、AnimateDiff → 又是1小时
  5. 调试环境报错、版本冲突 → 2小时

对，你没看错，光部署就花了4个多小时。中间还碰到 CUDA 版本跟 PyTorch 不匹配的问题，差点放弃。

但一旦跑起来，自由度确实无敌：

  • LoRA随便加：想要某种特定画风？去 CivitAI 下个 LoRA 模型加上就行。我下了个"吉卜力风格"的 LoRA，配合 SD 3.5 出的图真的太好看了
  • ControlNet精准控制：可以上传参考图的骨架/边缘/深度图来控制构图，这是MJ做梦都做不到的。你画个火柴人，它就按你的姿势生成
  • Inpainting局部修改：出了一张图但某个角落不满意？圈出来让它重画那个区域就行，其他地方不动
  • 批量生成：写个workflow节点流，晚上挂着让它跑，第二天早上收获几百张图慢慢挑

显存需求参考：

分辨率最低显存推荐显存
512x5124GB6GB
768x7686GB8GB
1024x10248GB12GB
1024x1024 + ControlNet10GB16GB

缺点也不少：

  • 学习曲线陡峭，特别是 ComfyUI 的节点系统，一开始看着跟意大利面一样
  • 出图速度慢，1024x1024 在我的 4070Ti 上要 15-20 秒
  • 社区资源虽然丰富但鱼龙混杂，下了几个 LoRA 效果完全不是预览图那样
  • 模型文件巨大，我现在光模型就占了 80 多 GB 硬盘

适合人群：有独立显卡的技术宅、想深度定制画风的创作者、需要大量出图的人。

DALL·E 3 —— 最懒人友好的选择

DALL·E 3 最大的优势就是零门槛——你已经在用 ChatGPT 了，直接在对话框里描述需求就能出图，不用额外注册任何东西。

而且ChatGPT会帮你"优化"提示词。你随便写个"画一只猫"，它会自动展开成详细的描述。这对不擅长写prompt的人特别友好。

实际体验：

  • 优点：跟ChatGPT对话式交互非常自然，可以一轮一轮地调整，“把背景换成蓝色”、“猫的表情再萌一点”
  • 优点：对复杂指令的理解力不错，“一个戴着圣诞帽的柯基坐在火箭上飞向月球"这种抽象需求它能理解
  • 缺点：画质跟MJ比差了一个档次，特别是人物的精细度和光影质感
  • 缺点：每天出图有数量限制（Plus用户大概40-50张），不够用
  • 缺点：风格偏"数字插画”，很难出写实风格的图

有意思的是，DALL·E 3 在文字渲染上做得不如 Flux，但比 MJ 和 SD 好。简单的英文字母基本能写对，中文嘛……还是算了。

适合人群：偶尔需要配图的普通用户、不想折腾的人、已经是ChatGPT Plus用户的人。

Flux 是今年最大惊喜

之前朋友推荐我试 Flux，我还不以为意。心想又是一个蹭热度的新工具吧。结果一试就惊了——这画质水平，放半年前说是 MJ 出的我都信。

Flux 有三个版本：

  • Flux Schnell：开源免费版，速度快画质一般
  • Flux Dev：开源进阶版，画质很好
  • Flux Pro / Pro 1.1：商用版，画质最好，API 付费

我主要用的是 Flux Pro 1.1，通过 Replicate 或者 Together AI 的 API 调用。

真正让我惊艳的几个点：

  1. 文字渲染能力碾压全场。让它在图里写字居然能写对——不光是英文，连中文都有不错的成功率。这一点MJ和SD到现在都做得很差
  2. 人物真实感。生成的人物照片，不仔细看真的分不出是AI生成的。皮肤纹理、毛孔、头发丝都有
  3. 性价比高到离谱。通过 API 调用大概 3-5 毛钱一张图，比 MJ 便宜太多了
  4. 提示词遵从性好。你说什么它画什么，不会自作主张加"MJ美学滤镜"
  5. 本地部署也可以。Flux Dev 可以在本地用 ComfyUI 跑，12GB 显存就行

当然缺点也有——Flux没有MJ那种"调性"。MJ出的图有一种说不清道不明的"高级感"，像是有个专业摄影师帮你调过色。Flux出的图更"老实"，你给什么prompt它出什么图，不会帮你美化。这一点见仁见智。

适合人群：追求性价比的人、需要文字渲染的人、API调用做自动化的开发者。

中文提示词国产工具占优

这个必须要展开讲。

海外工具你得用英文写提示词，写得好不好直接影响出图质量。你以为简单翻译一下就行？不是的。

比如中文提示词"古风少女"，你翻译成"ancient style girl"，出来的图可能是穿着希腊长袍的欧洲女孩。你需要写成"Chinese traditional hanfu girl, Tang dynasty style, ink painting aesthetic"才能得到你想要的效果。

通义万相就没这个问题。你写"晚霞中的古风少女站在竹林里弹琵琶"，它是真的能理解每个词的意思，不会给你搞成赛博朋克竹林、金属琵琶之类的抽象组合。

通义万相的优缺点：

  • 优点：中文理解碾压海外工具，免费额度大方，出图速度快（云端计算）
  • 优点：对中国文化元素理解到位——旗袍、汉服、水墨画、书法这些都画得像回事
  • 缺点：整体画质跟MJ还是有差距，特别是光影细节
  • 缺点：风格选择比较少，不太适合欧美/科幻类风格

文心一格：试了之后感觉比通义万相差了一个档次。中文理解还行，但出图质量不太稳定，同一个prompt跑10次可能只有一两次满意。而且它的"审美"偏那种……怎么说呢，像淘宝首页？不是不好看，就是太"商业"了。

提示词写作技巧（实战经验）

用了这么多工具，总结了几条实用的提示词技巧：

1. 结构化描述

好的提示词应该包含这几个要素：主体 + 场景 + 光影 + 风格 + 技术参数

“一个穿着汉服的少女（主体），坐在荷花池边的凉亭里（场景），黄昏的逆光打在她脸上（光影），中国水墨画风格（风格），8K分辨率，虚化背景（技术参数）”

2. 善用否定提示词

告诉AI你"不要"什么有时候比告诉它你"要"什么更有效：

  • “no watermark, no text, no blurry, no extra fingers”
  • 在SD里可以用 negative prompt 专门写，效果很显著

3. 参考图比文字更高效

如果你脑子里有清晰的画面但文字描述不出来，大部分工具（SD的IPAdapter、MJ的--sref）都支持上传参考图。这比你绞尽脑汁写prompt高效10倍。

4. 多生成多挑选

AI出图本质上是"概率"游戏。不要指望一次出完美的图。我通常的做法是：同一个prompt跑10-20张，挑最好的一两张，然后再用inpainting微调不满意的局部。

成本对比（月度使用100张图的情况）

工具月费100张图大概成本额外成本
MJ Basic$10/月$10超出额度另算
MJ Standard$30/月$30无限relaxed模式

如果你只是偶尔用用，通义万相的免费额度完全够了。如果是重度用户，MJ Standard + 本地SD的组合最划算。

版权和商用注意事项

这个很多人忽略了，但其实很重要：

  1. MJ：免费版生成的图不能商用。10刀版小规模商用OK，但需要标注AI生成
  2. SD：开源模型生成的图基本没有版权限制，最自由
  3. DALL·E 3：生成的图归用户所有，可以商用
  4. 通义万相/文心一格：免费额度内生成的图通常有使用限制，商用需要看条款
  5. Flux：Schnell和Dev模型用Apache 2.0协议，商用友好；Pro需要通过API付费

建议：如果涉及商用场景，仔细看各工具的使用条款，别踩坑。

我的AI绘画工作流

分享一下我目前稳定使用的工作流：

  1. 博客头图/封面：Flux Pro API 生成底图 → Photoshop 简单调色裁剪。成本低、速度快、质量够用
  2. 社交媒体配图：通义万相（中文场景） + DALL·E 3（随手生成），免费为主
  3. 需要精细控制的图：本地 SD 3.5 + ComfyUI + ControlNet，花时间但效果最好
  4. 重要项目/商用场景：MJ Standard 出多个版本 → 挑最好的 → SD Inpainting 微调

最后

AI 绘画现在已经不是"能不能用"的问题了，而是"选哪个用"的问题。根据自己的需求和预算选就行，没有最好的只有最合适的。

一点感慨：两年前我还需要去 Unsplash 翻半天才能找到一张凑合能用的配图，现在描述一下就能生成完全贴合主题的图。这个进步速度真的挺吓人的。照这个趋势，再过两年是不是连摄影师都不需要了？

当然，工具再好也只是工具。你的审美、创意和表达能力才是真正的壁垒。AI可以帮你把脑子里的画面变成图片，但"想到什么画面"这件事，目前还是只有人类能做。
